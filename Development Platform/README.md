ARM是一个企业的名字，也是移动端处理器的代名词，其原本含义：Advanced RISC Machine，即基于RISC（精简指令集）的高级机器。

ARM公司只负责架构和内核设计，并进行授权。

ARM现在已经发展出完备的产品体系了，刚好对应A、R、M三个系列：

A即Application，注重实现复杂应用，如手机处理器；

R即Real Time，注重可靠的实时性，常用于工业类的环境等；

M即Micro controller，微控制器，低成本高性能且应用最广泛。



# ARM Cortex-M系列

应用最广泛的就是STM32、NXP LPC和GD32三个系列。

1、Cortex-M3/4权威指南-来了解ARM的架构；
